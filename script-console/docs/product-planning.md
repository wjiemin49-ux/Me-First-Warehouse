# 本地离线脚本中控台产品规划

## 产品概述

Script Console 是一个本地优先、离线可用的 Windows 桌面中控台，用于统一发现、索引、启动、停止和监控 `D:\me\脚本` 下的各类脚本项目。

## 目标用户

- 需要集中管理多个自动化脚本的个人开发者
- 依赖长期运行本地任务、批处理、Python/Node/PowerShell 脚本的 Windows 用户
- 需要让 AI 生成的新脚本自动纳入治理的用户

## 信息架构

- 总览仪表盘
- 脚本列表
- 脚本详情
- 日志中心
- 设置
- 接入向导

## 交互原则

- 实时状态变化通过事件总线同步到前端
- 重要操作使用明确状态色和二次确认
- 列表与详情均支持快速跳转和反馈

## 风险与边界

- v1 仅支持 Windows
- 默认不执行危险 shell 拼接命令
- 兼容模式下的脚本允许手动启动，但默认不允许自动启动
